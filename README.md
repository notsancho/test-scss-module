## Welcome

## Getting started

## I. setup ENV

```
REACT_APP_API_URL=
```

## II. install deps

```
yarn
```

- if you don't have yarn, install it using `npm i -g yarn`

## III. start project

```
yarn start
```

### III.I. start storybook

```
yarn storybook
```

## IV. build project

```
yarn build
```
